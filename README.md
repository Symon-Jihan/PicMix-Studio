# PicMix Studio

Online photo mixing/editing web app.

## Features
- Editor Lab: adjust color/tone, presets, and download
- Collage Studio: build collages with layouts, frames, text overlay, and export quality
- Tone Fusion: apply style-based tone mapping with live preview
- Fully local: runs in your browser (no logins, no uploads)

## Tech Stack
- HTML, CSS, Vanilla JavaScript
- Canvas API

## How to Run Locally
1. Download/clone this repo.
2. Open `index.html` in your browser.
3. (Optional) Use a local server (e.g. VS Code Live Server) for best results.
